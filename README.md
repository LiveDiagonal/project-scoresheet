# Project Scoresheet

Open-source Haskell library for modeling baseball and working with [Retrosheet event data](http://www.retrosheet.org/eventfile.htm).

## Getting started

```
stack build
stack exec print-box-scores testgame.txt
```

## Running the tests

```
stack test
```
